# my-bini-app

A Bini.js app — zero-config React framework.

## Quick Start

```bash
pnpm install
pnpm run dev
```

## Production

```bash
pnpm run build
pnpm start        # served by bini-server
```

## Static Export (GitHub Pages / Netlify Static / S3)

```bash
pnpm run export   # vite build --mode export
```

Generates a fully pre-rendered `dist/` with per-route `index.html` files and a smart `404.html` — no server required.

## Linting & Formatting

```bash
pnpm run lint     # Oxlint — 50-100x faster than ESLint
pnpm run format   # Oxfmt  — Prettier-compatible formatter
pnpm run check    # lint + format check combined (great for CI)
```

## API Routes

Create files in `src/app/api/`. Export a Hono app:

```typescript
// src/app/api/hello.ts
import { Hono } from 'hono'

const app = new Hono().basePath('/api')

app.get('/hello', (c) => c.json({ message: 'Hello!' }))

export default app
```

Access at `http://localhost:3000/api/hello`

## File Structure

```
my-bini-app/
├── src/
│   ├── app/
│   │   ├── api/              ← API routes (Hono)
│   │   ├── layout.tsx       ← root layout + metadata
│   │   ├── page.tsx         ← /
│   │   ├── not-found.tsx    ← custom 404
│   │   └── globals.css
│   └── main.tsx             ← mounts <App /> (auto-generated by bini-router)
├── public/
│   ├── favicon.ico
│   ├── apple-touch-icon.png
│   ├── og-image.png
│   └── site.webmanifest
├── .oxlintrc.json
├── .oxfmt.json
├── vite.config.ts
└── package.json
```

## Layouts

- `src/app/layout.tsx` — root layout, uses `{children}`, export `metadata` here
- Nested layouts (e.g. `src/app/dashboard/layout.tsx`) use `<Outlet />` from react-router-dom

## Powered by

- **Vite 8**       — Rolldown-powered builds, dramatically faster
- **bini-router**  — filesystem routing + API middleware
- **bini-export**  — static SPA export (GitHub Pages, Netlify, S3)
- **bini-overlay** — dev overlay badge
- **bini-env**     — environment file display
- **bini-server**  — production server (zero dependencies)
- **Oxlint**       — fast Rust-based linter (replaces ESLint)
- **Oxfmt**        — fast Prettier-compatible formatter
